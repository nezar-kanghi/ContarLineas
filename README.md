Ene estre ejercicio leemos un fichero de texto llamado datos.txt y contamos cuántas líneas contiene. Utilizamos bufferReader junto con fileReader para leer el archivo linea por linea, vamos incrementando un contador cada vez que readLine() devuelve una lñinea distinta de null. al terminar  muestra por consola el numero total de lineas del archivo. Al igual que en todos los ejercicios de ficheros controlamos posible excepcion mediante IOException
